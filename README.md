# Garbage Classification Project (Image Processing & Computer Vision)

## 📌 Project Overview

This project focuses on building a **garbage classification system** using image processing and computer vision techniques.  
The goal is to automatically classify waste images into meaningful categories to support better waste management and environmental sustainability.

The workflow includes:
- Data collection and merging
- Data preprocessing and understanding
- Image processing techniques
- Model training and evaluation
- Result analysis and reporting

## 🚀 How to Run the Project

1. Clone the repository:
```bash
git clone https://github.com/the-salma-ahmad/garbage-classification-project.git
````

2. Navigate to the project folder:

```bash
cd garbage-classification-project
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Open Jupyter Notebook:

```bash
jupyter notebook
```

5. Run the notebook inside the `notebooks/` folder.
